# This is a Health Index projects API

## Run the project

1. Clone the project
2. install the packages

```bash
npm install
```

3. Define the environment variables in a `.env` file as shown in the `.env.example` file
4. Run the project

```bash
npm start
```

## API Endpoints (modified the documentation\*\*\*\*)
